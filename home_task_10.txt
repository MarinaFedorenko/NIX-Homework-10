1. Дан массив double[]. Найти максимальное число, минимальное число, среднее
арифметическое и сумму. (рекомендация - создайте только один стрим для
решения задачи)
2. Дана коллекция лет (java.time.Year). Вернуть високосные годы в виде списка,
отсортированного по возрастанию.
3. Дан список строк. Вырезать из каждой строки участки, удовлетворяющие
регулярному выражению.
4. Дан список строк со словами, разделенными одним или более пробелами.
Вернуть список слов-палиндромов.
5. Сгенерировать бесконечную последовательность простых целых чисел (тип
числа - BigInteger). Вернуть список из n первых вероятных простых чисел
(точность вычисления простого числа должна быть не менее 1 - 2 -100 ).
6. Дан список дней рождения сотрудников фирмы (Collection<LocalDate>). Вернуть
количество дней рожденья в каждом месяце (Map<Month, Integer>).
7. Дан массив целых чисел. Разбить его на четные и нечетные, отсортировать и те,
и другие по убыванию.
8. Дан список унарных операторов UnaryOperator<T>. Объединить их в один
унарный оператор, который применяет их все.
